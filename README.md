# MI PRIMERA PAGINA

## Temas vistos en clase

### El Hardware
El hardware es la parte física del computador, es decir, todo lo que puedo tocar: el monitor, el teclado, el mouse, los cables, la memoria, el disco duro, la tarjeta madre, entre otros. Es el conjunto de componentes que hacen posible que el equipo funcione. Yo entiendo el hardware como el “cuerpo” del computador, ya que sin él no podríamos ejecutar ningún programa.

### Software
El software es la parte lógica del computador, lo que permite que el hardware sepa qué hacer. Son los programas, aplicaciones y sistemas operativos que usamos a diario, como Windows, Word o los navegadores. Yo lo veo como el “cerebro” que le da instrucciones al hardware para realizar tareas específicas.
El hardware y el software trabajan juntos para que un computador funcione correctamente. El hardware, por sí solo, no puede hacer nada si no tiene instrucciones; y el software necesita del hardware para ejecutarse.
Esta integración permite que todas las partes del sistema trabajen coordinadas para cumplir las tareas que el usuario solicita. Sin esa conexión, los programas no podrían funcionar y los dispositivos serían solo piezas sin utilidad.

### Lenguajes de Programación
Los lenguajes de programación son herramientas que usamos para comunicarnos con la computadora. Nos permiten darle órdenes paso a paso mediante un código que el sistema entiende. Algunos ejemplos que he aprendido son Python, Java y JavaScript. Gracias a ellos, se pueden crear programas, páginas web y aplicaciones que usamos todos los días.

### Fundamentos de Programación
Los fundamentos de programación son los conceptos básicos que nos permiten entender cómo se crean los programas. A través de ellos aprendí que programar es darle instrucciones precisas al computador para que realice una tarea. Estos fundamentos incluyen el uso de variables, operadores, estructuras de control y datos.
Comprendí que antes de escribir código, es importante analizar el problema, planear una solución y luego traducirla a un lenguaje de programación. Es como enseñar al computador paso a paso lo que debe hacer, de forma ordenada y lógica.

### Variables
Las variables son espacios en la memoria del computador donde se pueden guardar datos que pueden cambiar durante la ejecución del programa. Me gusta pensar en las variables como cajas con un nombre, donde puedo guardar números, textos o resultados de operaciones.
Por ejemplo, si guardo en una variable el número de estudiantes de un curso, puedo usar ese dato después para calcular promedios o hacer comparaciones. Entendí que las variables hacen que los programas sean dinámicos y puedan trabajar con diferentes valores.
Estructuras de Datos
Las estructuras de datos son formas de organizar y almacenar la información dentro de un programa. Sirven para manejar los datos de manera más eficiente. Entre las más comunes están las listas, arreglos, pilas, colas y diccionarios.
Aprendí que, dependiendo del tipo de problema, se elige la estructura de datos adecuada. Por ejemplo, una lista sirve para guardar varios valores ordenados, mientras que un diccionario me permite relacionar un dato con otro. Gracias a las estructuras de datos, los programas pueden manejar grandes cantidades de información sin desordenarse.


### Ejemplo básico de programa en Python 
```python
# Definimos variables
a = 25
b = 30

# Sumamos las variables
suma = a + b

# Mostramos resultados
print(f"El resultado de sumar {a} y {b} es {suma}")
```

### Reflexión personal
Aprender sobre el hardware, el software y los fundamentos de programación ha sido una experiencia muy enriquecedora para mí. Al principio, todos estos conceptos me parecían algo lejanos o difíciles de entender, pero poco a poco fui descubriendo que cada uno tiene un papel fundamental en el funcionamiento de los computadores y en el mundo tecnológico que nos rodea.
Me llamó mucho la atención comprender cómo el hardware y el software se complementan, porque antes pensaba que un computador funcionaba “por arte de magia”, y ahora entiendo que detrás hay todo un proceso lógico, físico y programado que lo hace posible. Saber que el hardware es como el cuerpo y el software como el cerebro me ayudó a visualizarlo mejor y a valorar el trabajo que hay detrás de cualquier dispositivo que usamos a diario.
También me pareció muy interesante conocer sobre los lenguajes de programación, ya que son una forma de comunicarnos con las máquinas. Aunque todavía me parecen un reto, me emociona la idea de que con el tiempo pueda crear mis propios programas o aplicaciones. Entender lo que son las variables, las estructuras de datos y los pasos lógicos que siguen los programas me hizo ver la programación como una herramienta poderosa para resolver problemas y para desarrollar la creatividad.
En general, estos temas me ayudaron a abrir la mente y a mirar la tecnología desde otra perspectiva. Antes solo veía un computador como una herramienta para estudiar o entretenerme, pero ahora lo veo como un sistema complejo que combina ciencia, lógica y diseño. Sé que todavía tengo mucho por aprender, pero me siento motivada a seguir explorando este campo porque cada nuevo conocimiento me acerca más al mundo digital del que hoy todos hacemos parte.
